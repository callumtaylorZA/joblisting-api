# Job listings API
Basic REST API used to manage job listings and applicants.

### Requirements
#### Swagger
Swagger-node

#### Node.js
Install nodejs
Install npm
Npm install (install project dependencies)

#### Other Dependencies
MySQL (npm mysql2)
dotenv

Create .env file in project root directory.
DB_HOST=host
DB_USER=user
DB_PASSWORD=secret
DB_DATABASE=db-name
DB_CONNECTION_LIMIT=conn-limit
PORT=port
